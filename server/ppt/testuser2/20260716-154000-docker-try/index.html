Docker = 应用的便携集装箱

一次构建，到处运行 — 把应用和环境打包在一起
🖼️

镜像 (Image)

应用的"蓝图"模板，包含操作系统、依赖、代码等所有内容。分层结构，只读不可变。

📦

容器 (Container)

镜像的运行实例，轻量、隔离、可启停。每个容器就是一个小型沙箱环境。

🏪

仓库 (Registry)

镜像的"应用商店"，Docker Hub 上有数百万个公共镜像，随拉随用。

💡 生活类比

镜像 → 📀 系统安装光盘（类），可以创建多个实例
容器 → 🖥️ 安装好的系统（对象），独立运行互不干扰
仓库 → 📱 应用商店，需要什么直接下载安装
Dockerfile → 📋 制作镜像的"配方说明书"
🐳 核心能力：轻量 · 隔离 · 可移植 · 一致

5 分钟，从 0 到第一个容器

安装 → 运行 → 部署，比你想象的简单
1
📥 安装 Docker
一行命令搞定（Linux/Mac/Windows 均支持）
2
✅ 验证安装
运行 hello-world 确认 Docker 就绪
3
🌐 运行 Nginx 服务
一条命令启动一个 Web 服务器
4
🧩 多服务一键启动
Docker Compose 定义完整应用栈
💡

Docker Desktop 提供图形界面，适合初学者。安装后打开即可开始。

# 1. 安装（Linux）
curl -fsSL https://get.docker.com | sh

# 2. 验证
docker run hello-world

# 3. 运行 Web 服务
docker run -d -p 8080:80 nginx:alpine
→ 访问 http://localhost:8080

# 4. 多服务（docker-compose.yml）
version: '3'
services:
  web:
    image: nginx:alpine
    ports:
      - "8080:80"
